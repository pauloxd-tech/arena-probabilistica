# Arena Probabilística

Arena Probabilística é um motor narrativo probabilístico criado como
estudo aplicado de tomada de decisão sob incerteza.

O sistema modela cenários onde recursos são limitados, eventos raros
têm impacto desproporcional e decisões precisam ser tomadas sem
previsibilidade total.

Embora apresentado como um motor de RPG, seus princípios são
diretamente aplicáveis a contextos como análise de risco, segurança
da informação, arquitetura de sistemas e tomada de decisão estratégica.

O projeto foi conscientemente encerrado após cumprir seu objetivo de
clareza e aprendizado estrutural.
